# ChatHub - Cloudflare实时聊天平台

基于Cloudflare Workers和Pages构建的现代化实时聊天应用。

## 功能特性

- 🚀 **实时通信** - 基于WebSocket的实时消息传输
- 💬 **多房间支持** - 支持多个聊天房间
- 👥 **在线用户列表** - 实时显示在线用户
- 📝 **消息持久化** - 使用Durable Objects存储
- 😊 **表情支持** - 丰富的表情符号
- 📱 **响应式设计** - 适配桌面和移动设备
- 🔔 **通知系统** - 桌面通知和声音提醒
- 🎨 **主题切换** - 支持多种主题
- 🔄 **自动重连** - 网络断开自动重连

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **后端**: Cloudflare Workers
- **存储**: Durable Objects
- **部署**: Cloudflare Pages + Workers
- **实时通信**: WebSocket
